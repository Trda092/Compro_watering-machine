Abstract

เนื่องจากหลายๆคนนั้น นิยมปลูกต้นไม้เล็กๆในห้อง แล้วบางครั้งเราอาจจะลืมรดน้ำต้นไม้เหล่านั้น
พวกเราจึงได้คิดค้นอุปกรณ์สำหรับรดน้ำต้นไม้ที่ควบคุมได้ผ่าน internet โดยสร้างจาก micro controller
มินิโปรเจ็คนี้จึงได้ระบบการควบคุมการรดน้ำต้นไม้ เพื่อความสะดวกสบายของผู้ที่ดูแลต้นไม้

Method

หลักการทำงานของระบบนี้ จะทำงานผ่านหน้าเว็บที่ใช้ควบคุม Controller ที่พัฒนาจาก Software arduino
ให้ไปควบคุมส่วนของมอเตอร์ที่จะปล่อยน้ำอีกทีหนึ่งโดยลิงค์ข้อมูลที่เชื่อมกับ Database ของ Firebase

ตัวอย่างการต่อบอร์ดที่จะใช้งาน


ขั้นตอนที่ 1 เราจะส่งสถานะการทำงานเป็นดิจิตอลบิตของปั้มน้ำผ่านเว็บแอพพิเคชั่นไปยัง Realtime database ที่ Firebase



ขั้นตอนที่ 2 ESP8266 (NodeMCU) จะรับค่าสถานะการทำงานจาก Realtime database ที่ Firebase





สถานะปิด(0) สถานะเปิด(1)





ขั้นตอนที่ 3 สั่งการ Relay ตามค่าสถานะที่ได้รับมาจาก Realtime database สถานะปิด(0) น้ำจะไม่ไหล สถานะเปิด(1) น้าไหล
